# Alura_Desafio_AluraStore
Desafío o proyecto realizado con el objetivo de hacer uso de la biblioteca de Python Pandas. El objetivo del proyecto fue ayudar a entender las bases de la biblioteca Pandas, así como aprender a analizar la información extraída durante el análisis de los resultados.
## Conocimientos Adquiridos 
Consolidar los conocimientos adquiridos durante el módulo de Python para Data Science, además de desarrollar pensamiento lógico y crítico al momento de analizar los resultados filtrados durante el desarrollo del código.
## ¿Cómo compilar el código?
En el repositorio están subidos los archivos o bases de datos de las 4 tiendas que se usaron en el proyecto, por lo que ya no es necesario hacer uso de la importación de los archivos desde Drive; en su lugar solo se necesita colocar el nombre o ruta de acceso de los archivos al momento de hacer la importación con la biblioteca Pandas.

El orden para compilar el código es lineal, por lo que módulo por módulo el código debe ser compilado.
