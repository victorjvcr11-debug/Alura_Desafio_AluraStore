# Alura_Desafio_AluraStore
Desafío o proyecto realizado con el objetivo de hacer uso de la biblioteca de Python Pandas. El obejtivo del proyecto fue ayudar a entender las bases de la biblioteca Pandas, asi como aprender a analizar la información extraida durante el analisis de los resultados.
## Conocimiento Adquiridos 
Consolidar los conocimientos adquiridos durante el modulo de Python para Data Science, ademas de desarrollar pensamiento logico y critico al moemnto de analizar los resultados filtrados durante el desarrollo del codigo.
## Compilar el codigo
En repositorio estan subidos los archivos o base de datos de las 4 tiendas que se usaron en el proyecto, por lo que solo ya no es necesario hacer uso de la importanción de los archivos desde drive, en su lugar solo se necesita colocar el nombre o ruta de acceso de los archivos al momento de hacer la importacion de la biblioteca Pandas. 

El orden para compilar el codigo es lineal, por lo que modulo por modulo el codigo debe ser compilado.
